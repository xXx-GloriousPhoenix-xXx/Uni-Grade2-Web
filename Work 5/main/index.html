If not given: If it's present in local storage - stored is used. If it's not present in local storage - denies content setting.

Row - current row number (Indexing responds natural numbers). When filled, shows in text area content of current row. If not given - Denies content setting.

Set - Updates quantity of rows in table. Assigns text area content to current row of table in local storage.

Gen - Creates table from local storage data and displays in block-2.
